# Friendly AI AEP Tool Documentation

This directory contains the documentation for the Friendly AI AEP Tool, built with [Docusaurus](https://docusaurus.io/).

## Getting Started

To run the documentation site locally:

```bash
cd docs
pnpm install
pnpm start
```

This will start a development server at `http://localhost:3000`.

## Building

To build the documentation for production:

```bash
pnpm build
```

The static files will be generated in the `build` directory.

## Structure

- `docusaurus.config.js` - Main configuration file
- `sidebars.js` - Sidebar navigation configuration (to be created)
- `docs/` - Markdown documentation files (to be created)
- `blog/` - Blog posts (optional, to be created)
- `src/` - Custom React components and pages (to be created)
- `static/` - Static assets like images (to be created)
